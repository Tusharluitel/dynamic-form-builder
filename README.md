# Dynamic Form Builder

**Dynamic Form Builder** (DFB) is a Drupal 7 based to create forms, question. Also view the responses and audit the data in the application.

- Drupal - version 7.103
- PHP - version 8.2.30
- Solr - version 9.10.1
